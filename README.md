# AI视频创作平台
## 平台简介
- 本平台为一站式全流程生成AI视频创作平台，输入1个主题 → 输出1部专业级长视频（含分镜/画面/配音/剪辑）
- 功能定位：基于开源技术栈的本地化部署AI视频创作平台，实现从主题到视频的全流程自动化生产。
- 功能优势
1. AI智能生成文案、分镜、视频场景、角色动作等。
2. 丰富的视频模板：主题创意、AI读书、儿童绘本等，适配不同场景专业视频。
3. 提供写实、卡通、电影、科幻等多种艺术风格选择。
4. 内置多种AI语音，支持文本转语音功能，实现自然流畅的配音解说。
5. 提供多种视频比例选择，完美适配抖音、B站、小红书、今日头条等主流视频平台。
6. 支持1080P高清视频导出，满足不同平台的发布需求。

## 开源技术
 1. 开源了自研AI视频创作平台
 2. 集成N8N智能体流程调度平台
 3. 集成大语言模型：Ollama+DeepSeek
 4. 集成ComfyUI+Z-Images生图模型
 5. 集成通义万相Wan2.2视频模型
 6. 集成Index-TTS(2.0)语音合成模型

## 平台特点
 1. 开源本地化部署
  - 采用开源技术栈+本地部署，无需依赖外部服务，数据安全可控

## 安装指南
1. 环境准备
   - Python 3.11
   - Git
   - FFmpeg
   - Moviepy
   - MySQL

2. 克隆项目
```bash
git clone https://github.com/assen0001/aivideo.git
cd aivideo
```

3. 安装依赖（建议用uv或conda环境部署）
```bash
pip install -r requirements.txt
```

5. 配置环境
  - 把.env.example文件复制一份，改名为：.env
  - 配置AIVIDEO_URL为本项目访问链接（必填，需要映射到公网访问，建议用https）
  - 本地部署ComfyUI + z-Images + Wan2.2，并在.env中配置COMFYUI_URL(源码部署要自己安装，镜像部署容器中已安装好)
  - 本地部署Index-TTS(2.0)语音合成模型，并在.env中配置INDEXTTS_URL(源码部署要自己安装，镜像部署容器中已安装好)
  - 用aivideo_db.sql文件创建数据库，修改.env中数据库配置信息(源码部署要自己安装，镜像部署容器中已安装好)
  - 企业版部署N8N智能体调度平台。(个人版省略本步骤，企业版须独立部署N8N工作流平台)
  - 企业版部署Ollama + DeepSeek。(个人版省略本步骤，企业版可独立部署或调用其他大语言模型API)
  - 部署过程中有任何问题可以联系我们提供技术支持和部署服务。
  - 也可以直接运行已部署好镜象：
   - 仙宫云镜像地址：https://www.xiangongyun.com/image/detail/5bd7a57f-d8a2-42e4-b25f-5399e6903976
   - CSDN镜像地址：https://ai.csdn.net/?openMirrorId=img695c7c081ee71b11f194b1c3

## 使用说明
1. 启动服务
```bash
python main.py
```

2. 访问平台
   - 打开浏览器访问 http://localhost:5800

3. 基本流程
   - 输入主题/视频类型/视频风格/配音等参数
   - 生成文案脚本
   - 生成字幕语音
   - 生成分镜画面
   - 合成最终视频
   - 下载或分享视频
   - 可一键生成视频，也可以分步骤生成

4. 高级功能（部分开发中）
   - 批量生成：支持同时处理多个主题
   - 模板定制：可自定义文案生成模板
   - 风格调整：可修改视频风格参数
